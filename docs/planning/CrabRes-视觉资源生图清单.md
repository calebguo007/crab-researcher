# CrabRes 视觉资源生图清单

> 风格统一：3D 卡通渲染（类似 Pixar/Nintendo 质感），圆润、可爱但不幼稚
> 背景：透明 PNG
> 用途：Web 界面、分享卡片、Landing Page、App 图标

## 二、10 种海洋生物（核心！每种需要多个状态）

**统一风格要求：**
- 3D 卡通渲染，柔和光影
- 大眼睛（Pixar 风格），眼白清晰，黑色瞳孔带反光点
- 身体圆润饱满，线条流畅
- 每只大约占画面 80%，居中
- 透明背景 PNG
- 每只需要 3 个视角：正面（默认）、3/4 侧面、正侧面
