# CrabRes 视觉体系 v2 — 角色与 UI 重新定义

> **核心转向**：从"海洋动物"到"不是明确动物但让人想抱的抽象生物"
> 
> 参考：宝可梦/数码宝贝/洛克王国的造物逻辑 + 毛毡/黏土的材质感
> 
> 更新时间：2026-04-02

---

## 一、为什么要改

### 旧方案的问题
- 10 种海洋动物（螃蟹、章鱼、水母...）太具体、太常见、没记忆点
- 用户不会说"我的水母今天在研究竞品"——没有情感连接
- 看起来像儿童教育 App，不像有实力的增长工具

### 新方向的灵感
- **洛克王国冰淇淋生物**：蛋形身体 + 食物元素混搭 = 不知道是什么但很可爱
- **绿色毛毡外星人**：毛茸茸质感 + 大眼睛 + 极简五官 + 配饰（厨师帽+厨具）= 让人想捏
- **宝可梦/数码宝贝**：不是真实动物，是"类生物"——有自己的物种设定但不存在于现实

### 2026 设计趋势支撑
1. **包容性抽象**：不定义物种/性别，让所有用户都能产生投射
2. **微表情系统**：同一角色多种情绪状态（Duolingo 路线）
3. **Plushcore / 毛毡质感**：手工感 > 完美渲染，传递温度和信任

## 二、CrabRes 生物体系重新定义

### 核心设定
不叫"海洋动物"了。叫 **CrabRes Spirits（增长精灵）**。

每个精灵是一种"不存在于现实世界的可爱小生物"，和用户的产品类型绑定，但不是字面上的动物——是抽象的、有个性的、让人想收藏的角色。

### 10 种精灵（替代旧的 10 种海洋动物）

| 旧 | 新精灵名 | 形态灵感 | 配色 | 对应产品类型 |
|---|---------|---------|------|------------|
| 螃蟹 | **Pix** | 圆形小精灵，头顶一个小闪电触角，微笑大眼 | 品牌橙 #F97316 | SaaS / 工具类 |
| 章鱼 | **Lumi** | 发光水滴形，身体半透明，能看到里面的小星星 | 紫色 #8B5CF6 | 社区 / 社交类 |
| 水母 | **Mochi** | 软糯团子形，像大福，头顶冒蒸汽 | 青色 #06B6D4 | 内容 / 媒体类 |
| 河豚 | **Puff** | 毛茸茸球形，鼓鼓的腮帮子，小短手 | 金色 #EAB308 | 电商 / 零售类 |
| 海马 | **Sage** | 蘑菇形头部，戴小眼镜，手拿迷你书 | 绿色 #10B981 | 教育 / 学习类 |
| 海星 | **Spark** | 星形身体但五角是圆润的，头顶有画笔触角 | 粉色 #EC4899 | 创意 / 设计类 |
| 海龟 | **Terra** | 方形身体圆角，像一块带表情的积木，稳重感 | 深绿 #059669 | 金融 / 稳健型 |
| 小丑鱼 | **Bop** | 蝌蚪形，永远在弹跳，表情夸张开心 | 红色 #F43F5E | 娱乐 / 游戏类 |
| 鲸鱼 | **Cosmo** | 云朵形，大眼睛微微向上看，有梦幻感 | 蓝色 #3B82F6 | 企业 / B2B |
| 小虾 | **Pip** | 最小的精灵，种子形，有一双坚定的大眼睛 | 粉红 #FB7185 | 超早期 / 刚开始 |

### 材质风格
统一风格：**Needle Felting (羊毛毡) × 3D Render**

```
关键词：wool felt texture, soft matte finish, visible fiber texture,
round body, minimal limbs, big glossy black eyes with white reflection,
tiny smile, pastel background, studio lighting, isometric view
```

不是光滑的塑料 3D，是摸得到纤维质感的毛毡渲染。
这个风格：
- 天然有"手工感"→ 传递温度
- 不完美的纤维纹理 → 区别于 AI 光滑渲染
- 色彩柔和 → 适合长时间看的工具界面
- 极度可爱 → 让人想截图分享

### 每个精灵的状态（微表情系统）
每个精灵需要 6 种表情：
1. **happy** — 开心微笑，眼睛弯弯
2. **thinking** — 戴上小眼镜，歪头，一只手托腮
3. **working** — 戴小安全帽/耳机，认真表情
4. **excited** — 跳起来，星星眼，手举高
5. **worried** — 缩小身体，小汗滴，眼睛向上看
6. **sleeping** — 闭眼，Z-z-z，身体微微发光

## 四、UI 图标重新设计

不用 emoji，也不用线性图标。用 **3D 毛毡质感小物件** 作为图标。

| 图标 | 形态 | 用途 |
|------|------|------|
| 搜索 | 毛毡放大镜 | 研究功能 |
| 团队 | 3 个小精灵排成一排 | 专家圆桌 |
| 写作 | 毛毡羽毛笔 + 纸 | 执行/文案 |
| 增长 | 毛毡上升箭头（柔软弯曲的） | 增长指标 |
| 验证 | 毛毡盾牌 + 勾 | 策略审核 |
| 通知 | 毛毡小铃铛 | 提醒 |
| 设置 | 毛毡齿轮 | 设置 |
| 分享 | 毛毡纸飞机 | 分享功能 |
| 闪电 | 毛毡闪电 | 发现/洞察 |
| 日历 | 毛毡日历翻页 | 内容日历 |

## 五、UI 风格升级

### 从"普通卡片"到"动态玻璃态"

核心 CSS 变化：

```css
/* 玻璃态按钮 */
.btn-glass {
  background: rgba(255, 255, 255, 0.08);
  backdrop-filter: blur(20px) saturate(180%);
  -webkit-backdrop-filter: blur(20px) saturate(180%);
  border: 1px solid rgba(255, 255, 255, 0.15);
  border-radius: 16px;
  color: white;
  padding: 12px 24px;
  font-weight: 600;
  transition: all 0.3s ease;
  position: relative;
  overflow: hidden;
}

.btn-glass::before {
  content: '';
  position: absolute;
  top: 0;
  left: -100%;
  width: 100%;
  height: 100%;
  background: linear-gradient(90deg, transparent, rgba(255,255,255,0.1), transparent);
  transition: left 0.5s;
}

.btn-glass:hover::before {
  left: 100%;
}

.btn-glass:hover {
  background: rgba(255, 255, 255, 0.15);
  border-color: rgba(255, 255, 255, 0.3);
  transform: translateY(-2px);
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.12);

/* 玻璃态卡片 */

}
```
