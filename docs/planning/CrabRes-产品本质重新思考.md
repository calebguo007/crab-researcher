# CrabRes 产品本质重新思考

## 一、为什么我们之前想的都太粗了

每条增长路径背后，是一整个行业的 know-how：

| 增长路径 | 背后是什么 | 涉及多少专业 |
|---------|----------|------------|
| 找博主种草 | MCN 公司 / PR 公司 | 数据选号、背调、brief 写作、物流、追踪、结算 |
| 广告投放 | 代理商 / 投放优化师 | 素材创意、受众定向、出价策略、归因分析、A/B 测试 |
| 社媒运营 | 新媒体公司 | 平台算法、内容策划、视觉设计、社群管理、数据分析 |
| SEO | SEO 公司 | 关键词研究、技术 SEO、内容生产、外链建设、算法追踪 |
| PR/媒体 | 公关公司 | 记者关系、新闻角度、危机处理、媒体采购 |
| 展会 | 活动公司 | 展位设计、物料制作、现场执行、后续跟进 |

每一个格子里，是 **一家公司里好几个人** 在做的事。

我们想用一个 Agent 替代所有这些人？

**不可能。至少现在不可能。**

---

## 二、那 CrabRes 到底应该是什么？

### 不是替代执行团队

我们不是要替代那 5 个做博主种草的人。
我们也不是要替代广告投放优化师。
我们也不是要替代 SEO 专家。

### 是替代那个做决策的人

在一个公司里，有一个人（通常是 CMO/VP Growth/创始人自己）需要：

1. **看清全局**：我的产品适合走哪些增长路径？
2. **做优先级决策**：预算有限，先做哪个？为什么？
3. **知道每条路的深度**：找博主不是"找博主"三个字，是 22 步 SOP
4. **追踪结果**：做了以后效果怎么样？哪个路径 ROI 最高？
5. **动态调整**：数据说 Reddit 效果 3 倍于 LinkedIn，砍 LinkedIn 加 Reddit

**这个人需要的不是执行力，是判断力 + 知识 + 数据。**

**CrabRes = AI CMO（首席营销官）**

不是做事的人，是做决策的人。但他的决策比人类 CMO 好在哪？
- 基于实时数据而不是直觉
- 研究过竞品的真实策略而不是猜
- 知道 100 种增长方法的 SOP 而不是只懂自己擅长的 3 种
- 24/7 盯着数据不休息

## 三、Demo 阶段的"卧槽点"

### 不需要做到的
- ❌ 自动帮你发帖（OAuth 复杂、封号风险高）
- ❌ 自动帮你投广告（涉及真金白银，责任太大）
- ❌ 自动帮你联系博主（社交关系不能自动化）
- ❌ 覆盖所有增长路径的全部细节

### 必须做到的（用户说"卧槽"的点）

**1. 知道用户不知道的事实**
> "你的竞品 X 的 Instagram 60% 流量来自 3 个特定博主，他们的合作费在 $200-500 之间，联系方式在 bio 里。"

用户反应："我花了一周也没找到这些信息，你 30 秒就给我了。"

**2. 给出用户没想到的路径**
> "你做 AI 项链，别先做 Instagram。数据显示你的目标用户 35% 在 Pinterest 上搜索 'smart jewelry'。Pinterest 的 CPC 是 Instagram 的 1/3。这是你的蓝海。"

用户反应："我从没考虑过 Pinterest，但这个数据说服我了。"

**3. 每条路径给到"第一步就能做"的深度**
> "Pinterest 策略第 1 步：注册 Business Account → 申请 Rich Pins → 用这个关键词列表创建 10 个 Board → 每天 Pin 5 张产品图（尺寸 1000×1500）。这是你第一周的内容日历。"

用户反应："不用我想，照着做就行。"

**4. 指出一个让用户冒汗的风险**
> "⚠️ 你的定价 $299 是竞品 ORRA 的 2 倍，但你没有他们的品牌背书。如果不解决信任问题，广告投多少都是浪费。优先做 3 件事建立信任：..."

用户反应："其他工具都在夸我，只有这个告诉我真正的问题。"

### 核心：不在于做了多少，在于每一个输出都让用户感觉"这不是 AI 在忽悠我，这是一个真正懂行的人在帮我"

## 四、知识深度问题怎么解决

### 问题
LLM 的通用知识不够深。它知道"可以找博主合作"，但它不知道：
- 蝉妈妈和飞瓜的数据怎么看
- 什么样的完播率算好的
- 博主报价的行业标准是多少
- 怎么判断一个博主是不是买的粉

### 解决方案：知识工程，不是 prompt 工程

```
层级 1：通用知识（LLM 自带）
  "博主合作是一种增长策略"
  → 太浅，人人都知道

层级 2：结构化知识（Playbook 模板）
  "博主合作分 9 步：定义画像 → 搜索 → 分层 → ..."
  → 我们刚做完这个。比通用知识好，但每一步的细节还不够

层级 3：领域专家知识（需要积累）
  "Instagram 博主选号的 5 个核心指标：
   1. 互动率 > 3%（不是看粉丝量）
   2. 评赞比 > 0.05（低于这个说明互动注水）
   3. 粉丝画像重合度（用 Modash 查，和你的目标用户重合 > 60%）
   4. 历史合作频率（每月 > 4 条广告的博主粉丝信任度低）
   5. 内容调性匹配（你的产品放在他的 feed 里不违和）"
  → 这种知识不是 LLM 能凭空生成的，需要注入

层级 4：实时数据知识
  "博主 @xxx 粉丝 23K，互动率 4.2%，最近 30 天无负面，
   上次合作品牌是 YYY（$300 一条帖子），邮箱在 bio 里"
  → 需要实时爬取 + 数据库
```

### 对应的技术实现

| 层级 | 实现方式 | 优先级 |
|------|---------|--------|
| L1 通用 | LLM 自带 | ✅ 已有 |
| L2 结构 | Playbook 模板 + skills_registry | ✅ 刚做完 |
| L3 领域 | **知识注入**：按垂直领域整理专家级知识，写成结构化文档，注入专家 prompt | 🔜 下一步 |
| L4 实时 | **数据管道**：爬取 + API 接入（博主数据库、广告数据、SEO 数据） | 📋 V2 |

### L3 怎么做（最关键的下一步）

不是让 LLM 猜，是 **人类整理好知识，让 LLM 引用**。

类比：医疗 AI 不是让 GPT 猜怎么治病，是把医学指南注入进去让它引用。

CrabRes 也一样：
- 整理"博主合作"的 **行业标准 SOP**（从真实的 MCN 公司、PR 手册中提取）
- 整理"广告投放"的 **平台规则手册**（每个平台的尺寸、规则、算法偏好）
- 整理"社媒运营"的 **平台算法指南**（每个平台推荐逻辑是什么）
- 整理"SEO"的 **2026 最新规则**（Google 算法变化、AI Overview 影响）

这些知识写成结构化文档，存在 `skills_registry.py` 或独立的知识库中。
专家被调度时，自动加载对应领域的知识。

**每多整理一个垂直领域的专家级知识，CrabRes 就在这个领域更"靠谱"。**

## 五、关于"搜广推"算法

你的直觉完全对。CrabRes 本身也需要：

### 搜（Search）
用户描述产品 → CrabRes 需要从知识库中精准匹配：
- 最相关的增长路径
- 最相似的历史案例
- 最匹配的 Playbook 模板

实现：product embedding → 和知识库做向量检索
我们已经有 pgvector，这个可以做。

### 推（Recommendation）
基于用户画像 + 产品类型 + 预算 + 历史实验数据，**主动推荐** 下一步：

"今天你应该做的事：
1. 给 @blogger_A 发 follow-up 邮件（3 天前发的初次邮件，还没回）
2. 在 r/fashiontech 发一篇帖子（你上次的帖子效果最好的时间段是周二上午）
3. 检查 Pinterest Ad 的 A/B 测试结果（已经跑了 72 小时）"

这不是 LLM 实时推理出来的——是基于 **用户的执行进度 + 时间线 + 历史效果数据** 算出来的。

### 广（Matching/Ranking）
未来如果有社区 / Playbook 市场：
- 做 AI 项链的用户 → 推荐做过智能手环的用户的成功 Playbook
- 预算 $500 + SaaS 产品 → 推荐预算相近的成功案例

## 六、Demo 阶段的聚焦策略

### 不要做
- 不要试图覆盖所有增长路径
- 不要试图自动执行
- 不要试图做实时数据管道

### 要做
1. **选 2-3 个增长路径做到"深到震惊"**
   - Reddit/社区增长（最通用、最容易验证效果）
   - KOL/博主种草（最容易展示差异化）
   - 一个"X vs 竞品"对比路径（展示研究深度）

2. **在这 2-3 个路径上，把 L3 领域知识做到位**
   - 不是 LLM 泛泛而谈，是注入了真实的行业标准
   - 用户能感觉到"这不是 AI 在猜，这是有真功夫的"

3. **把"真实研究"作为核心体验差异化**
   - 每次输出都基于 Tavily 搜索的真实数据
   - 能说出竞品名字、真实价格、真实流量数据
   - 这是 ChatGPT 做不到的

4. **把"追踪闭环"作为留存差异化**
   - 用户做了 → 贴回链接 → CrabRes 追踪效果 → 给出调整建议
   - 这是一次性工具做不到的

## 七、一句话重新定义

> **CrabRes = 你的 AI CMO。**
> **不是帮你做事，是帮你做对的决策 + 给你执行手册 + 追踪效果 + 迭代优化。**
> **它比你聪明（因为研究了竞品数据），比你勤快（24/7 盯着），比你诚实（会告诉你不想听的真话）。**
